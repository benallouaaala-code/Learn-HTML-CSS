أهم وسوم CSS

تعلم أساسيات CSS من خلال الخصائص الأساسية والأكثر استخداماً في تصميم صفحات الويب.

الخاصية الوصف المثال
color لون النص color: red;
background-color لون الخلفية background-color: blue;
font-size حجم الخط font-size: 16px;
margin الهامش الخارجي margin: 10px;
padding الهامش الداخلي padding: 10px;
border الحدود border: 1px solid black;
width العرض width: 100px;
height الارتفاع height: 100px;
